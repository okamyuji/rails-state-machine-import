# サンプルデータ

アプリをすぐに試すためのダミーデータです。

## 使い方

### シードデータの投入

各状態のサンプルジョブをまとめて投入します。

```bash
bin/rails db:seed
```

投入後、`http://localhost:3000` のトップ画面に以下のジョブが現れます。

- 顧客マスタ取込（初回登録直後） ... registered 状態でモーダル表示
- 商品マスタ取込（モーダルとオーバーレイ競合） ... importing_modal_queued 状態でオーバーレイ優先
- 在庫データ取込（インポート中） ... importing 状態
- 注文履歴取込（一時停止中） ... paused 状態
- 売上データ取込（完了） ... completed 状態
- 取引先データ取込（失敗） ... failed 状態（エラーメッセージ付き）
- 未登録のひな形ジョブ ... idle 状態

### CSVサンプル

- `customers.csv` ... 10件の顧客データ
- `products.csv` ... 10件の商品データ

CSVはアプリ内で直接読み込む機能は持たせていません。新規登録時の「対象行数」入力の参考値として使ってください。たとえば `customers.csv` なら対象行数 10 と入力します。
